## 📖 Sobre o Projeto

O **Desafio dos Namorados** é uma aplicação web criada para transformar perguntas sobre o relacionamento em uma experiência divertida e envolvente.

A ideia é simples:

1. Um dos parceiros cria ou responde às perguntas.
2. O outro tenta acertar as respostas.
3. No final, o sistema calcula a pontuação e mostra o resultado.

Perfeito para:
- 💘 Dia dos Namorados
- 🎂 Datas especiais
- 🎉 Brincadeiras entre casais
- 📱 Compartilhar com amigos e familiares
